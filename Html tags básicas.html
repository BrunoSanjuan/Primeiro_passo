Tags básicas do HTML: O início

O HTML, línguagem de marcação, usado para o desenvolvimento de páginas web, contém algumas tags simples para que você, caro leitor, caso não conheça, possa saber como funciona, por trás das telas, nas codificações o funcionamento da tela que você vê quando acessa uma página web.

Neste conteúdo, vamos falar apenas do básico, do html puro, sem incluir estilos, beleza visual para a página, que seria preciso o uso do CSS conjuntamnete.

A estrutura inicial do HTML contém as tags !DOCTYPE, html, head, meta, title e body. As tags são declaradas dentro de " < > ".
Por exemplo: < p >

Analise e veja, que você, caro leitor, consegue visualizar apenas um título e alguns parágrafos nesta página web, pois, como inicialmente dito, começaremos do início, básico do básico.
E você sabe quais tags foram usadas para que este título e parágrafos estejam visíveis e estruturado para você?

Para título Pricipal ou sub-título do texto, usamos as tags < h1 >, < h2 >, < h3 >, < h4 >, < h5 > ou < h6 >.
E os parágrafos?

Bem para o parágrafo ou novos parágrafos usamos a tag < p >.
E para quebras de linhas, assim como exatamente nesta linha, sem que necessite de um novo parágrafo?

Para quebras de linhas, como em alguns pontos deste texto e geralmente usados em poemas, usamos a tag < br >.

Por fim, aprendemos 3(três) tags básicas para se utlizar e seu funcionamento no html e sabemos que sem elas, não teríamos uma estrutura para que tudo fique organizado em sua tela e seja visto deste modo agrádavel, iremos nos aprofundar um pouco mais nos próximos tópicos e utilizar novas tags para chegar ao final de uma boa estrutura finalizada de html.
